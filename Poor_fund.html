Predict Poor Performing Funds using Linear Regression

One should not invest in a fund which is consistantly giving bad returns over years. We gathered past 10 years data from morningstar website and built a Linear Regression model to predict the fund which have poor performance.

By using Linear Regression to Anlayze fund data for over 1200 Mutual funds, we predicted 155 mutual funds which are giving poor performance. We have randomly selected few poor performing frunds from the set of predicted 155 funds. We will show their performance for current year using Returns data from Yahoo Fiance Data.
